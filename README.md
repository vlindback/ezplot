# ezplot

## Description

Command line tool for plotting charts from data from files.

Written in Rust for educational purposes.

## Installation

## Usage

You can use ezplot to plot data from a file to the terminal.

### Plotting a file

    ezplot <file_path>

    ezplot <file_path>

## Supported Files

Currently supported:

Planned:

* csv
* json
* xml
* txt

Not happening (but would be nice)

* xlxs
* odt
